# m-page-lab

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
页面通常为实验性、演示性或临时性用途，不针对任何特定域名或网站。

## 项目简介

本仓库旨在集中存放各类独立 HTML 文件，便于版本管理、历史追溯和分享。  
每个页面可视为一个独立的小项目，彼此之间无依赖关系。

## 目录说明

- `pages/` — 存放所有 HTML 页面文件，按需可进一步划分子目录。
- `assets/` — 页面用到的公共资源（如样式、脚本、图片等），非必需。
- `docs/` — 相关文档或说明文件，可选。

## 页面归档说明

- 每个 HTML 文件应尽量自包含，减少对外部资源的依赖。
- 如需引用外部资源，建议使用相对路径或注明来源。
- 文件命名建议清晰、简短，便于识别用途。
- 鼓励在文件内添加简要注释，说明页面目的或运行方式。

## 维护说明

- 欢迎通过 Pull Request 提交新的独立页面，或对现有页面进行改进。
- 提交前请确保页面可正常打开，无明显错误。
- 不建议在页面中包含任何形式的营销、推广或诱导点击内容。
- 本仓库仅作为技术存档与展示，不提供在线托管或持续集成服务。
